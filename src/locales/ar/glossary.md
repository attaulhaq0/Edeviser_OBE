## Translation Notes

- **MSA (Modern Standard Arabic)**: Use formal Arabic suitable for educational contexts, not colloquial dialects.
- **Gender**: Use masculine forms as default (e.g., "الطالب" for student); context will clarify when feminine forms are needed.
- **Technical Terms**: Preserve English acronyms (ILO, PLO, CLO, XP, CQI) in parentheses when first introduced.
- **Consistency**: Always use the same Arabic term for the same English concept across all translation files.
- **Localization**: Dates, numbers, and currency follow `ar-QA` locale conventions (Arabic-Indic numerals, Hijri calendar references where applicable).

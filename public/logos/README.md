# Edeviser Logo Files

Save your logo images here:

- `logo1-icon-white-bg.png` — Gradient icon, white background
- `logo2-dark-bg-text.png` — Gradient icon + "E DEVISER" text, dark background (PRIMARY)
- `logo3-white-black-bg.png` — White icon + text, black background
- `logo4-black-white-bg.png` — Black icon + text, white background
- `logo5-icon-dark-bg.png` — Gradient icon only, dark background

## Usage
- `logo2` is the primary logo used across the app (login, sidebars, public pages)
- `logo1` / `logo5` are icon-only variants for favicons and small spaces
- `logo3` / `logo4` are for print and monochrome contexts
